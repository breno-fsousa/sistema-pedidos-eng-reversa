# Proposta de Arquitetura

## Modelo Proposto: MVC

### Camadas

### Model
- Produto
- Pedido
- Cliente
- Categoria

### View
- Interface web
- Telas de pedidos

### Controller
- Processamento de requisições
- Regras de negócio


## Separação de Responsabilidades

- Model → dados
- View → interface
- Controller → lógica


## Componentes Principais

- Sistema de pedidos
- Sistema de carrinho
- Sistema de autenticação (opcional)
- Integração com pagamento


## Benefícios

- Melhor manutenção
- Escalabilidade
- Código organizado
